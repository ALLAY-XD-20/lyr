DOCUMENTATION

Everything you need to get started

🚀 QUICK START

Invite Lyrixa to your server, ensure it has the correct permissions (Connect + Speak in voice channels), then run your first command:

+play <song name or URL>

🎵 MUSIC COMMANDS

Core music commands to control playback in your server voice channels. Lyrixa supports YouTube, Spotify, and SoundCloud links.

+play +pause +resume +skip +stop +queue +loop +seek +volume

🎛️ FILTER COMMANDS

Apply real-time audio effects to transform your music experience. Stack multiple filters for unique sounds.

+bassboost +nightcore +8d +vaporwave +slowdown +filter reset

📋 PLAYLIST SETUP

Create personal or server-wide playlists. Save your favourite tracks and load them instantly at any time.

+pl create <name> +pl add <song> +pl play <name> +pl list

⚙️ CONFIGURATION

Customize Lyrixa's behaviour for your server. Set DJ roles, default volumes, and channel restrictions.

+config djrole @Role +config prefix + +config maxvol 150

❓ GETTING HELP

Need more detail? Every command has its own help page. Use the help flags to explore all options.

+help +help play +command -h

PRIVACY POLICY

Lyrixa values your privacy above all else

1. DATA STORAGE

We do not store any user or server data. Lyrixa does not use a persistent database to track your personal information, messages, or history.

2. REAL-TIME PROCESSING

All data processed by Lyrixa is handled in real-time memory and is immediately discarded after the request is fulfilled or the bot session ends.

3. NO DATA SHARING

Since we never store any data, we have nothing to sell, share, or trade with third parties. Your server's activities remain completely private.

4. THIRD-PARTY SERVICES

Lyrixa may use third-party APIs (like Spotify or SoundCloud) to fetch music. These services only receive the name of the song you are searching for. No personal information is ever transmitted.
